## 2. 用户群与行为路径

### 2.1 小红书上的核心人群

| 人群 | 典型内容 | 他们想解决什么问题 | 当前信号强度 |
|---|---|---|---|
| 省事型家长 | `辅导作业`、`不费妈`、`孩子数学辅导` | 快速把孩子作业和错题问题处理掉 | 高 |
| 口语 / 英语自学用户 | `免费练口语`、`雅思口语对练` | 低成本、高频练习英语表达 | 高 |
| 追求提分和复盘的学生 | `错题本`、`查缺补漏`、`打印错题` | 把错题沉淀下来继续练 | 中高 |
| AI 工具横评用户 | `对比`、`哪个好用`、`正确率高不高` | 在多个 AI 工具之间做选择 | 中高 |

### 2.2 用户在小红书上展示出来的行为路径

| 行为路径 | 代表内容 | 说明 |
|---|---|---|
| 被“免费 + 省事”吸引 | `这可能是目前最高效的口语练习方法 免费！！！` | 免费是传播门槛最低的切入点 |
| 先用来救急 | `辅导作业`、`拍题答疑`、`一分钟学会 AI 学语文` | 决策入口是当前问题，不是长期规划 |
| 好用后转向复盘 | `学习就是查缺补漏`、`免费错题整理功能` | 错题本和打印功能让产品从一次性工具走向复盘工具 |
| 出错后迅速转负面 | `哪个AI正确率高呢`、`反正最让我失望的AI就是豆包` | 老师感建立得越强，错题带来的反噬越大 |

## 3. 功能认知与场景分布

### 3.1 被反复提及的功能主题

| 功能主题 | 命中量 | 用户怎么理解它 | 对正式报告的意义 |
|---|---|---|---|
| 口语练习 | `24` | 免费、陪练、可长期坚持 | 它不仅是解题工具，也在吃英语练习场景 |
| 作业辅导 | `24` | 帮家长省事、减少陪写作业摩擦 | 家长场景是真实获客入口 |
| 家长视角 | `23` | 不费妈、省心、能陪孩子学 | 说明小红书上的传播不只面向学生 |
| 竞品对比 | `22` | 与 DeepSeek、作业帮、夸克等对比 | 小红书是用户做轻决策的对比场 |
| 错题本 | `17` | 查缺补漏、打印、复盘 | 错题本是复盘资产层，不只是附属功能 |
| 拍照答疑 | `6` | 快速开始、立刻有反馈 | 这是流量入口，但不是全部价值 |
| 准确率 | `5` | 不准、会改答案、容易误导 | 这是小红书最敏感的负面触发点 |

### 3.2 小红书用户最在乎的不是“功能多”，而是“问题有没有立刻解决”

小红书里真正带动互动的内容，往往不是完整的功能说明，而是几类明确承诺：

- `这能不能马上把题讲清楚`
- `这能不能让我少辅导一次作业`
- `这能不能让我免费练口语`
- `这能不能把错题整理成后面还能继续用的东西`

这说明平台上的传播逻辑非常任务导向，不是品牌导向。

## 4. 正向反馈与负向反馈

### 4.1 正向反馈矩阵

| 主题 | 代表原话 / 内容 | 洞察 |
|---|---|---|
| 免费 | `免费练口语`、`零成本`、`不用会员也可以用` | 免费是扩散速度最快的传播钩子 |
| 讲题 / 陪练 | `最高效的口语练习方法`、`再次给大家演示豆包雅思口语对练` | 用户愿意展示“AI 带着练”的过程 |
| 家长减负 | `辅导作业再也不废妈`、`孩子数学辅导问题全搞定` | 家长人群买单的不是技术，而是负担减少 |
| 错题复盘 | `学习就是查缺补漏`、`免费错题整理功能` | 错题本让产品更像长期学习工具 |

### 4.2 负向反馈矩阵

| 主题 | 代表原话 / 内容 | 风险 |
|---|---|---|
| 准确率不稳 | `豆包搜题100分得了50分`、`一会儿答案a，一会儿答案b` | 一旦讲错，会直接破坏老师感 |
| 功能稳定性 | `一对一讲题功能时有时无` | 核心功能不稳定会削弱信任 |
| 输入方式受限 | `没有键盘输入，图书馆用不了` | 某些环境下使用门槛偏高 |
| 品牌混淆 | 用户分不清 `豆包`、`豆包老师`、`豆包爱学` | 会让传播心智分散，影响独立品牌建立 |

## 6. 这批数据能支持什么洞察

1. 豆包爱学在小红书上不是“深教研品牌”，而是“能立刻帮你把问题解决掉的学习助手”。
2. 小红书最强的传播点不是 `拍照答题` 本身，而是 `免费 + 讲得像老师 + 家长省事 + 口语能练`。
3. 用户开始形成对 `错题本` 的复盘认知，这使它从一次性问答工具向长期学习工具迈进。
4. 准确率是最关键的负面开关。老师感越强，出错时的负面反馈越尖锐。
5. 小红书是豆包爱学最重要的轻决策平台之一，适合看功能爆点、真实使用理由和口碑断裂点。
