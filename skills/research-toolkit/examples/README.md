# Examples 索引

这里放的是可直接打开查看的示例与 benchmark。

目标不是给“抽象模板”，而是给下载者一个明确上限：

- 高标准成稿长什么样
- 写作计划长什么样
- 体验报告怎么和正文联动
- 图证怎么进入章节

## 当前案例库

### `doubao-aixue-benchmark/`

这是当前最接近目标标准的一份 benchmark。

它会尽量保留这些内容：

- 主报告
- 写作计划
- 体验报告
- 审核门禁
- 事实核查
- 可视化计划
- 正文实际引用到的关键截图与图表

说明：

- 这是案例库，不是运行时输入目录
- 可以精简 raw 数据，但要尽量保留阅读和复盘所需的关键文件
- 如果某份案例成为新的 benchmark，应在这里新增目录，而不是覆盖旧案例
