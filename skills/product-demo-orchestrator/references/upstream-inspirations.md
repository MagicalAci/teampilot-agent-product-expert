# Upstream Inspirations

这个 skill 的结构和方法主要参考了以下开源仓库：

- `AvdLee/SwiftUI-Agent-Skill`
  - 参考点：`SKILL.md + references/` 的专题化拆分方式，以及 SwiftUI 场景下按需加载的知识组织方式
- `nextlevelbuilder/ui-ux-pro-max-skill`
  - 参考点：design system、anti-patterns、pre-delivery checklist 的思路
- `iOfficeAI/AionUi`
  - 参考点：demo 结果要可预览、可演示、可交接的产品闭环意识
- `affaan-m/everything-claude-code`
  - 参考点：质量门、交付打包、演示材料组织思路
- `ibelick/ui-skills`
  - 参考点：baseline UI、accessibility、metadata、motion performance 这类最后一公里打磨模型

本 skill 以“方法复用与结构借鉴”为主，不直接镜像这些仓库的完整实现。
