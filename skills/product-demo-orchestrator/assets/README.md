# Assets

这些模板用于初始化 demo 交付目录：

- `demo-brief-template.md`：冻结 demo brief
- `design-system-master-template.md`：生成 design system 主文件
- `preflight-checklist-template.md`：预演前检查清单
- `demo-walkthrough-template.md`：演示话术与路径
- `developer-handoff-template.md`：研发交接说明
- `demo-readme-template.md`：demo 目录说明
- `web-demo-template.html`：Web demo 入口
- `h5-demo-template.html`：H5 demo 入口
- `swiftui-*.swift`：SwiftUI demo 入口与 token 示例
