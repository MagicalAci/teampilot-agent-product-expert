# Demo Preflight Checklist

## 1. 可演示性

- [ ] 首页或首屏能在 5 秒内说明产品是什么
- [ ] 关键 CTA 或主路径清晰可见
- [ ] walkthrough 与实际页面一致

## 2. 视觉与信息层次

- [ ] 主标题、辅助信息、动作按钮层级清楚
- [ ] 颜色数量可控，没有风格漂移
- [ ] 设计系统与页面实现一致

## 3. Accessibility

- [ ] 交互元素具备可读标签
- [ ] 焦点状态可见
- [ ] 文本与背景对比度可接受

## 4. Metadata 与分享准备

- [ ] 页面标题明确
- [ ] 演示入口文件可识别项目名称
- [ ] 交接文档写清运行与展示方式

## 5. Motion Performance

- [ ] 仅对 `transform` / `opacity` 做轻量动画
- [ ] 默认时长控制在 150-300ms
- [ ] 有减少动效策略

## 6. 交接

- [ ] 已知限制已记录
- [ ] 后续研发接手点已记录
- [ ] zip 打包结果可用
