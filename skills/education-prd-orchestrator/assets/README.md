# Assets 索引

这里放的是可复用模板，不放具体任务证据。

## 当前资产

- `prd-template.md`
  - 新建 PRD 时的最小骨架
- `index-log-template.md`
  - 产品索引里的工作日志和版本记录模板
- `user-checkpoint-template.md`
  - 向用户请求补材料时的标准模板
- `diagram-brief-template.md`
  - 给配图 agent 的标准 brief
- `html-page-template.html`
  - 单文件页面稿模板，用于页面 / 功能 HTML 交付
- `chart-template.html`
  - 单文件图表模板，用于数据论证和截图交付
- `search-plan-template.md`
  - 搜索问题、关键词和批次计划模板
- `evidence-log-template.csv`
  - 证据登记表模板，用于把来源和判断显式绑定

## 使用原则

- `assets/` 放模板，不放案例证据
- 案例去 `examples/`
- 规则去 `references/`
