# Assets 说明

这里放的是可复用的表现层资产。

原则：

- 放模板，不放某个案例独有证据
- 放版式，不放具体任务判断
- 放共用规则，不放临时草稿

## 当前内容

- `chapter-visual-blocks.md`
- `writing-plan-table-template.md`
- `chapter-ops-tables.md`

## 适合继续放进来的内容

- 中文图注模板
- 章节图板样式约定
- 统一边框、宽度、留白规范
- 可复用的小型 SVG 模板

## 如果你想让团队“照表跑”

优先使用这两个模板：

- `writing-plan-table-template.md`
- `chapter-ops-tables.md`

前者负责章节合同和推进状态。
后者负责配图和章节事实校验。

## 不适合放进来的内容

- 某个产品独有截图
- 某次任务的录屏抽帧
- 某份报告的正文片段

这些应进入 `examples/` 或具体任务目录，而不是 `assets/`。
