# Examples 索引

这里放的是这个 Skill 的调用示例，不放真实业务证据。

## 推荐先看

1. `example-invoke.md` — 四条主命令如何唤起
2. `api-cookbook/README.md` — ★ API 能力积木库（可运行的真实 API 调用代码）
3. `example-script-command.md`
4. `example-test-command.md`
5. `example-validation-command.md`

## 作用区分

### 命令调用示例

- `example-invoke.md`
  - 说明四条主命令如何唤起
- `example-script-command.md`
  - 说明 `/AI脚本` 怎样明确要求脚本、README、调用文档一起交付
- `example-test-command.md`
  - 说明 `/AI测试` 怎样明确要求真实调用与测试报告
- `example-validation-command.md`
  - 说明如何用本 Skill 内置脚本做初始化、校验和打包

### API 能力积木库

- `api-cookbook/` — **11 个独立可运行的 Python 示例**
  - 覆盖：文本生成、流式、深度思考、续写、缓存、多轮对话、图片理解、视频理解、视觉定位、结构化输出、多 Agent 编排
  - 用途：在 `/AI脚本` 开发时按需组合这些积木搭建项目专用的 AI 脚本
  - 详见 `api-cookbook/README.md`
